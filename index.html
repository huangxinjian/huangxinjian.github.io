# 目标系统化

针对某一个具体的问题，形成详细或固定的流程化处理方案，下次再做同样的事情的时候，就安心按照这个流程执行即可，而不是在恐惧下毫无作为，担心这个不会，哪个不会，让时间白白流失。

例如我们想学习一门新技术，这门技术可能是你从来没有接触过的，就可以按照这样的流程去学习：

  1. 先搞清楚这门技术是因为什么原因而存在的，即用来解决什么样的问题的？
  2. 搞清楚这门技术需要什么样的基础知识，可以针对性的去阅读

# 五分钟策略

只要坚持5分钟，就可以去做别的事情，我们可以这样安慰自己，从而来进行学习。比方说打算看书，我们就跟自己约定好，只看5分钟，或者10 15 20分钟，坚持下来后就可以去做别的事。

# 持续培养习惯力

学会给我们的习惯打分：

  • 痛苦：某件事一旦我们不这样做，我就会有点痛苦
  • 热情：一旦我们这样做，我们会感到快乐
  • 时机：现在处于的环境，有利于养成我们的习惯
